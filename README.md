# test-codex-project

A small Tauri desktop demo that builds into a native executable.

## Scripts

- `npm install` installs the Tauri CLI for the project.
- `npm run dev` launches the desktop app in development mode.
- `npm run build` creates a release executable with `tauri build`.

## Build output

After a successful release build, the executable is written to:

- `src-tauri/target/release/test-codex-project`

On macOS, this is the native binary file you can run directly.
